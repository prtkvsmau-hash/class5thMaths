🔧 Google Sheets Setup Required

To enable automatic submission to Google Sheets, a teacher/admin must complete this one-time setup:

  1. Open Google Sheets and create a new spreadsheet.
  2. Go to Extensions → Apps Script and paste the Apps Script code (provided in the HTML comments at the bottom of this file).
  3. Click Deploy → New Deployment → Web App.
  4. Set Execute as: Me and Who has access: Anyone. Copy the Web App URL.
  5. Open this HTML file in a text editor, find APPS_SCRIPT_URL near the top of the <script> block, and paste the URL.
